Alles vorhandenen Videos zu den Veranstaltungen bis zum 14. April finden Sie auf [Sciebo](https://th-koeln.sciebo.de/s/CG7vMfpJBoAaTFT), das Passwort ist analog zu den Zoom-Räumen.
